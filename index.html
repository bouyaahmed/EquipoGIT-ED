Bienvenidos a Mi Página Web

Esta es una página sencilla creada como parte de un proyecto de desarrollo web que sirva como una práctica para aprender a utilizar el flujo de trabajo con pull requests en GitHub. El proyecto consistirá en una estructura básica de una página web, donde se crearán varias ramas de desarrollo y se enviarán cambios mediante pull requests para revisar, fusionar y gestionar el código en un repositorio.

Visita el sitio oficial de Egibide para más información.
